# wuziqi

## 介绍
使用go实现的五子棋游戏

## 功能
- 人人对战
- 人机对战
- 退出键
- 悔棋

## 运行
```shell
go run main.go
```

### release versions
- prime version : 机机对战，小棋盘，四字棋
- version 0.1: 人机对战，大棋盘，是否先手, 五子棋
- version 0.2: 文件整理，注释，优化
- version 0.3: 采用了go的GUI分支代码进行修改，文件整理，添加人机对战，开始菜单选择
- version 0.4: 添加了悔棋功能
- version 0.5: 基于alpha_zero的预训练模型增加了AI，目前棋盘尺寸更改为8x8
